2.3 Logics

Propositional Calculus (命题演算)

propositional variables $p, q, r, \cdots$

propositional constants true and false
(Well-formed) propositional formulas are constructed from the propositional variables and constants, using the unary connective negation ($\neg$) and the binary connectives disjunction ($\vee$) (析取), conjunction ($\wedge$) (合取), implication ($\to$), and equivalence ($\leftrightarrow$).

A truth assignment for a set $V$ of propositional variables is a function $\xi : V \to \{true, false\}$, the truth value is denoted as $\varphi[\xi]$.

$\varphi$ is satisfiable if at least one truth assignment makes it true; otherwise, it is unsatisfiable. It is valid if each truth assignment for the variables in $\varphi$ makes it true.

$\varphi$ logically implies $\psi$ (or $\psi$ is a logical consequence of $\varphi$), $\varphi \vDash \psi$, if for each truth assignment $\xi$, if $\varphi[\xi]$ is true, then $\psi[\xi]$ is true. $\varphi$ and $\psi$ are logically equivalent, dentoed as $\varphi \equiv \psi$, if $\varphi \vDash \psi$ and $\psi \vDash \varphi$.

literal: $p$ or $\neg p$. Conjunctive normal form (CNF): $\psi_1 \wedge \cdots \wedge \psi_n$, where $\psi_i$ is a disjunction of literals. Disjunctive normal form (DNF) : $\psi_1 \vee \cdots \vee \psi_n$, where $\psi_i$ is a conjunction of literals.

Theorem If $\varphi$ is a propositional formula, then there is some formula $\psi$ equivalent to $\varphi$ that is in CNF (respectively DNF).

First-Order Logic

first-order predicate calculus (一阶谓词演算)
Each first order language $L$ includes a set of variables, the propositional connectives, $\exists$ and $\forall$, ), (, and ,.

$L$ includes: 1. constant symbols; 2. $n$-ary predicate symbols; 3. $n$-ary function symbols.
propositional constants are true and false.
Terms built from constants, variables and function symbols.
Atom: true, false, and $R(t_1, \cdots, t_n)$ where $R$ is an $n$-ary predicate symbol, $t_i$ is a term.
A term or formula is ground if it involves no variables.
free and bound:

  1. variable in atom is free;
  2. $\varphi = \psi_1 \vee \psi_2$, $x$ in $\varphi$ is free if $x$ is free in $\psi_1$ or $\psi_2$;
  3. $\varphi = \exists y \psi$, $x \ne y$, $x$ in $\varphi$ is free if $x$ in $\psi$ is free, $y$ is bound. (each occurrence of $y$ is $\varphi$ that is free in $\psi$ is said to be in the scope of $\exists y$ at the beginning of $\varphi$)

An interpretation of a first-order language $L$ is a 4-tuple $\mathcal{J} = (U, \mathcal{C, P, F})$ where $U$: nonempty set of abstract elements called the universe (of discourse), $\mathcal{C, P, F}$ : mappings.

A variable assignment for $\varphi$ is a partial funciton $\mu : \text{variables of} L \to U$.
Sentence $\varphi$ logically imples sentence $\psi$, denoted as $\varphi \vDash \psi$, if each interpretation that satisfies $\varphi$ alse satisfies $\psi$.
