Chapter 1 Events and Probability

Definition (probability space) A probability space has three components:

  1. A sample space $\Omega$, which is the set of all possible outcomes of the random process modeled by the probability space;
  2. A family of sets $\mathcal{F}$ representing the allowable events, where each set in $\mathcal{F}$ is a subset of the sample space $\Omega$; and
  3. A probability function $\mathrm{Pr} : \mathcal{F} \to \mathbb{R}$.

Definition (probability function) A probability function is any function $\mathrm{Pr} : \mathcal{F} \to \mathbb{R}$ that satisfies the following conditions:

  1. For any event $E$, $0 \le \mathrm{Pr}(E) \le 1$;
  2. $\mathrm{Pr}(\Omega) = 1$; and
  3. For any finite or countably infinite sequence of pairwise mutually disjoint events $E_1, E_2, E_3, \cdots$,

When $E$ and $F$ are independent and $\mathrm{Pr}(F) \ne 0$, then $\mathrm{Pr}(E \mid F) = \frac{\mathrm{Pr}(E \cap F)}{\mathrm{Pr}(F)} = \frac{\mathrm{Pr}(E)\mathrm{Pr}(F)}{\mathrm{Pr}(F)} = \mathrm{Pr}(E)$.

This means that if two events are independent, then information about one event should not affect the probability of the second event.

Appilcation: Verifying Matrix Multiplication, i.e., to verify whether $AB = C$ efficiently

Algorithm($\Theta(n^2)$): Choose a random vector $\bar{r} = (r_1, r_2, \cdots, r_n) \in \{0, 1\}^n$. If $A(B\bar{r}) \ne C\bar{r}$, then return $AB \ne C$. Otherwise, return $AB = C$.

Theorem If $AB \ne C$ and if $\bar{r}$ is chosen uniformly at random from $\{0, 1\}^\star$, then $\mathrm{Pr}(AB\bar{r} = C\bar{r}) \le \frac{1}{2}$.
Proof: $D = AB - C \ne 0$ and $AB\bar{r} = C\bar{r}$ then it is known that $D\bar{r} = 0$, i.e., $\sum^n_{j=1}d_{1j}r_j = 0$, or equivalently, $r_1 = -\frac{\sum^n_{j=2}d_{1j}r_j}{d_{11}}$. ($D\ne 0$, 考虑$D$中包含非0元素的行，不失一般性，令$d_{11}\ne 0$).

According to the principle of deferred decisions, consider the situation $r_2, \cdots, r_n$ are chosen, then the right-hand side is determined ($-\frac{\sum^n_{j=2}d_{1j}r_j}{d_{11}}$), and there is at most one choice of $r_1$ that will make that equality holds.

The principle of deferred decisions corresponds to conditioning on the revealed values; when some of the random variables are revealed, we must condition on the revealed values for the rest of the analysis.

Application: A Randomized Min-Cut Algorithm

Min-Cut used in network reliability and clustering.
Algorithm: consists of $n-2$ iterations, in each, picks an edge in the graph and contracts that edge. (迭代过程中，新图的任一割集一定也是原始图的割集，但是原始图的每一个割集并不一定都是中间迭代过程中新图的割集)


Theorem The algorithm outputs a min-cut set with probability at least $2/n(n-1)$.
Proof: Let $E_i$ be the event that the edge contracted in iteration $i$ is not in $C$, where $C$ is a min-cut. And $F_i = \cap_{j=1}^i E_j$, $\mathrm{Pr}(E_i \mid F_{i-1}) \ge 1 - \frac{k}{\frac{k(n-i+1)}{2}} = 1 - \frac{2}{n-i+1}$ ($n - i$个点的图并且最小割的大小为$k$), $\mathrm{Pr}(F_{n-2}) = \mathrm{Pr}(E_{n-2}\cap F_{n-3}) \cdot \mathrm{Pr}(F_{n-3})$, $\mathrm{Pr}(E_1) = \mathrm{Pr}(F_1) \ge 1 - \frac{2k}{nk} = 1 - \frac{2}{n}$.
