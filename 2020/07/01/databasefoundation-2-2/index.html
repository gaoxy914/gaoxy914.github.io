2.2 Languages, Computability, and Complexity

alphabet $\Sigma$,
word over $\Sigma$ is $w = a_1,\cdots,a_n$, where $a_i \in \Sigma$,
length $|w| = n$,
concatenation (级联) of $u$, $v$ denoted $uv$, concatenation with itself $u^n$,
$\Sigma^*$ all words over $\Sigma$,

Finite Automata

acceptance, accept precisely the words that belong to some language of interest.
finite-state automata (fsa, 有穷状态自动机), scanning word, remembering only a bounded amount of information about what has already been scanned.
An fsa $M$ over $\Sigma$ is a 5-tuple $\langle S, \Sigma, \delta. s_0, F\rangle$:

  • $S$ finite set of states;
  • $\delta$ transition function $S \times \Sigma \to S$;
  • $s_0$ *start state$;
  • $F$ accepting states.

The fsa is only ALLOWED TO MOVE RIGHT.
A language accepted by some fsa is called a regular language.

Lemma(Pumping Lemma) For any regular language $L$, there exists an integer $n$ such that for all $x \in L$ with $|x| \ge n$, there exists $u, v, w \in \Sigma^\star$ such that $x = uvw$, and
  1. $|uv| \le n$,
  2. $|v| \ge 1$,
  3. for all $i \ge 0 : uv^iw \in L$.

For example, Let us prove $L = \{0^n1^n \mid n \ge 0\}$ is irregular using Pumping Lemma.

Assume that $L$ is regular, then by Pumping Lemma the above given rules follow.

Now let $x \in L$ and $|x| \ge n$. So by Pumping Lemma, there exists $u, v, w$ such that $x = 0^n1^n = uvw$ with $|uv| \le n$ and $|v| \ge 1$.
So $u = 0^a, v = 0^b, w = 0^c1^n$ where $a + b \le n, b \ge 1, c \ge 0, a + b + c = n$.

Then if $i = 0$, $uv^0w = uw = 0^a0^c1^n = 0^{a+c}1^n \notin L$, since $a + c \ne n$.

A regular expression over $\Sigma$ is wrriten using the symbols in $\Sigma$ and the operations concatenation, * and +.

A language $L$ is regular iff it can be specified by a regular expression.

Nondeterministic fsa (nfsa) accept the same set of language as fsa. However, the number of states in the equivalent deterministic fsa may be exponential in the number of states of the nondeterministic one.

Turing Machines and Computability

A language is recursively enumerable (r.e.) iff it is $L(M)$ for some TM $M$. It means that there is an algorithm that is guaranteed to say eventually yes on input $w$ if $w \in L$ but may run forever if $w \notin L$ (if it stops, it says no).

$L$ is recursive means that there is an algorithm that always decides in finite time whether a given word is in $L$. If $L = L(M)$ and $M$ always halts, $L$ is recursive. A language whose complement is r.e. is called co-r.e..
Some facts:

  1. If $L$ is r.e. and co-r.e., then it is recursive.
  2. $L$ is r.e. iff it is the domain of a computable function.
  3. $L$ is r.e. iff it is the range of a computable function.
  4. $L$ is recursive iff it is the range of a computable nondecreasing $(|f(w)| \ge |w|)$ function.

TIME($Ploy$) PTIME, SPACE($Poly$) PSPACE.

Membership in the class PTIME is often regarded as synonymous to tractability.

The set of recursive languages accepted by some NTM using time bounded by a polynomial is NP, space bounded by a polynomial is NPSPACE.
Savitch’s theorem states that PSPACE = NPSPACE
$L$ is reducible to $K$ if there is a computable mapping $f$ such that for each $w, w \in L$ iff $f(w) \in K$.
The complexity of the reduction function is lower than that for a class.
A recursive language is complete in $C$ by $C’$ reductions if for each $L \in C$ there is a function $f$ in $C’$ reducing $L$ to $K$.

NP-COMPLETE problem: 3-SAT
PSPACE-COMPLETE problem: QBF
PTIME-COMPLETE problem: CFG emptiness

Context-Free Grammars

A CFG is a 4-tuple $\langle N, \Sigma, S, P\rangle$, where:

  • $N$ a finite set of nonterminal symbols;
  • $\Sigma$ a finite alphabet of terminal symbols, disjoint from $N$;
  • $S$ a distinguished symbol of $N$, start symbol;
  • $P$ a finite set of productions of the form $\xi \to w$, where $\xi \in N$ and $w \in (N \cup \Sigma)^*$.

The language accepted by $G$: $L(G) = \{w \mid w \in \Sigma^*, S \Rightarrow^*(\text{transitive closure of} \Rightarrow) w\}.$

The specification power of CFGs lies between that of fsa's and that of TMs. All regular languages are context-free and all context-free languages are recursive.
Lemma(Pumping Lemma) For any $L$ in CFL, there exists an integer $n$ such that for all $x \in L$ with $|x| \ge n$, there exists $u, v, w, x, y \in \Sigma^\star$ such that $x = uvwxy$ and
  1. $|vwx| \le n$,
  2. $|vx| \ge 1$,
  3. for all $i \ge 0 : uv^iwx^iy \in L$.

Some questions about CFGs:
Decidable: emptiness [is $L(G)$ empty] finiteness [if $L(G)$ finite]

Undecidable: containment [is it true that $L(G_1) \subseteq L(G_2)$] equality [is it true that $L(G_1) = L(G_2)$]
