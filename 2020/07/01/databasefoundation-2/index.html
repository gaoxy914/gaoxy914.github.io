Theoretical Background

Binary Relation

A binary relation (二元关系) over a (finite or infinite) set $S$ is a subset $R$ of $S \times S$,
the crossproduct of $S$ with itself. $R(x, y)$, $xRy$ denite $(x, y) \in R$.

$n$-ary relation $S^n$.

A binary relation is reflexive (自反) if $(x, y) \in R$ for each $x \in S$;

A binary relation is symmetric (对称) if $(x, y) \in R$ implies that $(y, x) \in R$ for each $x, y \in S$;

A binary relation is transitive (传递) if $(x, y) \in R$ and $(y, z) \in R$ implies that
$(x, z) \in R$ for each $x, y, z \in S$.

A reflexive, symmetric, and transitive binary relation is an equivalence relation.
For each $x \in S$ the equivalence class $[x]_R = \{y \in S \mid (x, y) \in R\}$.

A partition of a nonempty set $S$ is a family of sets $\{S_i \mid i \in I\}$ such that:

  1. $\cup_{i \in I}S_i = S$,
  2. $S_i \cap S_j = \phi$, and
  3. $S_i \ne \phi$ for $i \in I$.

If $R$ is an equivalence relation on $S$, the family of equivalence classes over $R$ is a partition of $S$.

$E$ and $E’$ are equivalence relations on a nonempty set $S$, $E$ is a refinement of $E’$ if $E \subseteq E’$, i.e., for each $x \in S$, $[x]_E \subseteq [x]_{E’}$.
Each equivalence class of $E’$ is a disjoint union of one or more equivalence classes of $E$.

A partial order (偏序关系) of $S$ is a binary relation $R$ over $S$ that is reflexive, antisymmetric, and transitive.

A total order (全序关系) is a partial order $R$ over $S$ such that for each $x, y \in S$, either $(x, y) \in R$ or $(y, x) \in R$.

If $(S, R)$ is a partially order set, then a topological sort (拓扑序) of $S$ (relative to $R$) is a binary relation $R’$ on $S$ that is a total order such that $R’ \supseteq R$. (由某个集合上的偏序得到该集合上的一个全序，这个操作称之为拓扑排序)

$R$: a binary relation over $S$, $\mathbf{P}$: a set of properties of binary relations.
The $\mathbf{P}$-closure (闭包) of $R$ is the smallest binary relation $R’ \supseteq R$ and $R’$ satisfies all of the properties in $\mathbf{P}$.

Graph

A (directed) graph is a pair $G = (V, E)$, where $V$ is a finite set of vertexes and $E \subseteq V \times V$.

A directed path in $G$ is a nonempty sequence $p = (v_0, \cdots, v_n)$ of vertexes such that $(v_i, v_{i+1}) \in E$. An undirected path in $G$ is a nonempty sequence $p = (v_0, \cdots, v_n)$ of vertexes such that $(v_i, v_{i+1}) \in E$ or $(v_{i+1}, v_i) \in E$. The path is called proper if $v_i \ne v_j$ for $i \ne j$.

A (directed or undirected) cycle is a (directed or undirected) path $v_0, \cdots, v_n$ such that $v_n = v_0$. A cycle is proper if $v_0, \cdots, v_{n-1}$ is a proper path.

A graph $G = (V, E)$ is acyclic if it has no cycles or, equivalently, if the transitive closure of E is irreflexive.

Two vertexes $u, v$ are connected if there is an undirected path in $G$ from $u$ to $v$, and they are strongly connected if there are directed paths from $u$ to $v$ and from $v$ to $u$.
Connectedness and strong connectedness are equivalence relations on $V$. A (strongly) connected component of $G$ is an equivalence class of $V$ under (strong) connectedness. A graph is (strongly) connected if it has exactly one (strongly) connected component.

The distance $d(a, b)$ of $a, b$ is the length of the shortest path connecting $a$ to $b$.

The diameter (直径) of a graph $G$ is the maximum finite distance between two nodes in $G$.

A tree is a graph that has exactly one vertex with no in-edges, called the root, and no undirected cycles. A leaf of a tree is a vertex with no out edges. A forest is a graph that consists of a set of trees.

Let $G$ and $G’$ be two directed graphs. A function $h : V \to V’$ is a homomorphism (同态) from $G$ to $G’$ if for each pair $u, v \in V$, $(u, v) \in E$ implies $(h(u), h(v)) \in E’$. The function $h$ is an isomorphism (同构) from $G$ to $G’$ if $h$ is a one-one onto mapping from $V$ to $V’$. $h$ is a homomorphism from $G$ to $G’$, and $h^{-1}$ is a homomorphism form $G’$ to $G$. An automorphism (自同构) on $G$ is an isomorphism form $G$ to $G$.

Permutation

A permutation (置换) of $S$ is a one-one onto function $\rho : S \to S$.

A permutation $\rho’$ is derived from permutation $\rho$ by an exchange if the listing corresponding to $\rho$ and $\rho’$ agree everywhere except at some positions $i$ and $i + 1$, where the values are exchanged.

Theorem Given two permutations $\rho$ and $\rho'$, $\rho'$ can be derived from $\rho$ using a finite sequence of exchanges.
