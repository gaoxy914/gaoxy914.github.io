Coupon Collector’s Problem

coupon collector’s problem
$E[X] = nH_n$, where $H_n = \sum^n_{i = 1}1/n = \ln{n} + O(1)$.

Markov’s inequality $\mathrm{Pr}(X \ge 2nH_n) \le \frac{1}{2}$.

To use Chebyshev’s inequality, the variance of $X$ is needed. $X = \sum^n_{i=1}X_i$, $X_i$ are geometric random variables with parameter $(n - i + 1)/n$, and are independent $Var[X] = \sum^n_{i=1}Var[X_i]$.

Lemma The variance of a geometric random variable with parameter $p$ is $(1 - p)/p^2$.

$Var[X] = \sum^n_{i = 1}Var[X_i] \le \sum^n_{i = 1}(\frac{n}{n - i + 1})^2 = n^2\sum^n_{i = 1}(\frac{1}{i})^2 \le \frac{\pi^2n^2}{6}$.

Chebyshev’s inequality $\mathrm{Pr}(|X - nH_n| \ge nH_n) \le \frac{n^2\pi^2/6}{(nH_n)^2} = O(\frac{1}{\ln^2{n}})$.

The Randomized Linear Time Algorithm

Analysis

Theorem The randomized median algorithm terminates in linear time, and if it does not output FAIL then it outputs the correct median element of the input set $S$.

There are three “bad” events (line 6-7) such that the algorithm outputs FAIL.

  • $\varepsilon_1: Y_1 = |\{r\in R \mid r \le m\}| < \frac{1}{2}n^{3/4} - \sqrt{n} \Leftrightarrow l_d > n/2$; (the $(\frac{1}{2}n^{3/4} - \sqrt{n})$th smallest element of $R$ must be larger than $m$)
  • $\varepsilon_2: Y_2 = |\{r\in R \mid r \ge m\}| < \frac{1}{2}n^{3/4} - \sqrt{n} \Leftrightarrow l_d < n/2$;
  • $\varepsilon_3: |C| > 4n^{3/4}$.

Lemma $\mathrm{Pr}(\varepsilon_1) \le \frac{1}{4}n^{-1/4}$.

Proof: Let

$\mathrm{Pr}(X_i = 1) = \frac{(n - 1)/2 + 1}{n} = \frac{1}{2} + \frac{1}{2n}$.
The $\varepsilon_1$ is equivalent to $Y_1 = \sum^{n^{3/4}}_{i=1}X_i \le \frac{1}{2}n^{3/4} - \sqrt{n}$, that is $Y_1$ is the sum of Bernoulli trials, it is a binomial random variable with parameters $n^{3/4}$ and $1/2 + 1/2n$. Hence, $Var[Y_1] = n^{3/4}(\frac{1}{2}+\frac{1}{2n})(\frac{1}{2}-\frac{1}{2n}) < \frac{1}{4}n^{3/4}$.

Applying Chebyshev’s inequality $\mathrm{Pr}(\varepsilon_1) \le \mathrm{Pr}(|Y_1 - E[Y_1]| > \sqrt{n}) \le \frac{Var[Y_1]}{n} < \frac{1}{4}n^{-1/4}$.

Lemma $\mathrm{Pr}(\varepsilon_3) \le \frac{1}{2}n^{-1/4}$.

Proof: If $\varepsilon_3$ occurs, then at least one of the following two events occurs:

  • $\varepsilon_{3.1}$: at least $2n^{3/4}$ elements of $C$ are greater than the median;
  • $\varepsilon_{3.2}$: at least $2n^{3/4}$ elements of $C$ are smaller than the median.

If $\varepsilon_{3.1}$ occurs, it means that the order of $u$ in the sorted order of $S$ is at least $\frac{1}{2}n + 2n^{3/4}$ and thus the set $R$ has at least $\frac{1}{2}n^{3/4} - \sqrt{n}$ samples among the $\frac{1}{2}n - 2n^{3/4}$ largest elements in $S$. (如果$\varepsilon_{3.1}$发生，那就是$u$选的太大了，在中位数和$u$之间至少有$2n^{3/4}$个元素，那么在$S$中比$u$大的元素至多有$n - (\frac{1}{2}n + 2n^{3/4}) = \frac{1}{2}n - 2n^{3/4}$个，这些元素当中有至少$n^{3/4} - (\frac{1}{2}n^{3/4} + \sqrt{n}) = \frac{1}{2}n^{3/4} - \sqrt{n}$个被抽到，才会造成$u$太大了)

Let

$X = \sum^{n^{3/4}}_{i=1}X_i$ is a binomial variable with parameters $n^{3/4}$ and $\frac{1}{2} - 2n^{-1/4}$. Hence, $E[X] = \frac{1}{2}n^{3/4} - 2\sqrt{n}$ and $Var[X] = n^{3/4}(\frac{1}{2} - 2n^{-1/4})(\frac{1}{2} + 2n6{-1/4})$.

Applying Chebyshev’s inequality $\mathrm{Pr}(\varepsilon_{3.1}) = \mathrm{Pr}(X \ge \frac{1}{2}n^{3/4} - \sqrt{n}) \le \mathrm{Pr}(|X - E[X]| \ge \sqrt{n}) \le \frac{Var[X]}{n} < \frac{1}{4}n^{-1/4}$.

Theorem The probability that the randomized median algorithm fails is bounded by $n^{-1/4}$.

Expected Running Time: $T = p\times O(n) + (1 - p)\times T$, where $p = 1 - \frac{1}{n^4}$, $T = O(n)$.
